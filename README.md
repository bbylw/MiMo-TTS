# 🎙️ MiMo TTS - 智能语音合成

基于小米米墨 **MiMo-V2-TTS** 模型的在线语音合成工具，支持多种音色、格式和风格选择。

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deploy-brightgreen)](https://pages.github.com/)
[![Platform](https://img.shields.io/badge/Platform-Xiaomi%20MiMo-blue)](https://platform.xiaomimimo.com)

## 🔐 安全说明

本项目非常重视用户隐私和数据安全：

- ✅ **API 密钥本地存储**：密钥仅保存在您浏览器的 localStorage 中
- ✅ **不上传密钥**：密钥永远不会发送到任何第三方服务器
- ✅ **用户自主管理**：可随时清除已保存的密钥
- ✅ **开源透明**：代码完全透明，可自行审查

```
用户输入 API Key → 保存到本地浏览器 → 直接调用小米米墨 API
                                      ↓
                               不经过任何中间服务器
```

## 📖 使用指南

### 1. 配置 API 密钥

首次打开页面会弹出 API 密钥配置框：
1. 输入您的小米米墨平台 API 密钥
2. 点击"开始使用"
3. 密钥会自动保存在本地浏览器中

### 2. 生成语音

1. 在文本框中输入要转换的文字（最多 5000 字符）
2. 选择想要的**音色**（默认/中文女声/英文女声）
3. 选择音频**格式**（WAV/MP3/PCM）
4. （可选）选择**风格**（开心/悲伤/方言等）
5. 点击"生成语音"按钮
6. 等待生成完成后播放或下载

### 3. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 快速生成语音 |

### 4. 历史记录

最近 10 次生成的语音会自动保存在浏览器中，方便回听和重新下载。

## 🔧 API 参数说明

### 请求地址
```
https://api.xiaomimimo.com/v1/chat/completions
```

### 请求格式
```json
{
    "model": "mimo-v2-tts",
    "messages": [
        {
            "role": "user",
            "content": "请用自然流畅的语音朗读以下内容"
        },
        {
            "role": "assistant",
            "content": "<style>开心</style>明天就是周五了，真开心！"
        }
    ],
    "audio": {
        "format": "wav",
        "voice": "mimo_default"
    }
}
```

### 响应格式
```json
{
    "choices": [{
        "message": {
            "audio": {
                "data": "base64编码的音频数据"
            }
        }
    }]
}
```

## 💡 常见问题

### Q: 语音生成失败怎么办？

1. 检查 API 密钥是否正确
2. 确认网络连接正常
3. 文本长度不要超过 5000 字符
4. 尝试刷新页面重新输入密钥

### Q: 为什么风格没有生效？

部分风格可能需要特定内容配合才能有明显效果，建议：
- 使用风格对应的示例文本
- 尝试不同的文本长度

### Q: 如何获取 API 密钥？

1. 访问 [小米米墨开放平台](https://platform.xiaomimimo.com)
2. 注册/登录账号
3. 进入控制台 → API Keys
4. 创建新的 API Key

### Q: 需要付费吗？

**当前限时免费**！小米米墨 TTS 服务目前免费使用。
