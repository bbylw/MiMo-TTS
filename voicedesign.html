mimo-v2.5-tts-voicedesign

该模型仅使用 user 中的音色描述塑造声音。全新界面将提示、示例与输入区整合为一体，提高 prompt 设计效率。

建议 1-4 句，聚焦年龄/质感/情绪/语速四个维度。

口音可直接写入音色描述，如“带北方口音的普通话”。

避免混响、回声、EQ、压缩等后期处理词。

播放与下载

请先在首页保存 API Key（localStorage: mimo_api_key）。

如何写好音色描述（voice design prompt）

描述越具体、越生动，生成音色越贴近预期。推荐按“核心维度 + 场景补充”组织。

关键维度

  • • 性别与年龄：young woman in her mid-20s / 五十多岁的中年男性
  • • 音色质感：deep and gravelly / 丝滑醇厚、带着磁性
  • • 情绪语气：warm and confident / 温柔但带一丝疲惫
  • • 语速节奏：slow and deliberate / 语速极快，像连珠炮

可选增强维度

  • • 角色人设：narrator / podcast host / 深夜电台 DJ
  • • 说话风格：casual and colloquial / 一本正经地
  • • 场景描写：nature documentary / 给投资人路演
  • • 年代参照：1940s film noir / 八十年代译制片配音

示例（简洁）

Heavy Russian accent, gruff middle-aged male, blunt and matter-of-fact.

示例（专业）

一位年迈老先生，说带北方口音的普通话，语速缓慢沉稳，嗓音略带沙哑和沧桑感，像在讲故事。

  • • 避免冲突描述：如“稚嫩童声 + CEO 气场”。
  • • 避免模糊词：如“普通的、正常的、外国的”。
  • • assistant 的合成文本应匹配音色设定，效果更好。
